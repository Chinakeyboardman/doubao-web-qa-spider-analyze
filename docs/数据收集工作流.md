## 🎯 **数据收集操作流程**

### **步骤1：Query豆包并截图**
```
输入：Query文本
操作：
1. 在豆包中输入query
2. 等待回答生成
3. 截图保存（完整回答+商品清单）
4. 记录到表1、表2、表5
```

### **步骤2：搜索候选链接**
```
基于Query和推荐商品，在各平台搜索：

平台列表：
✅ 小红书（图文A）
✅ 抖音（图文A）
✅ 知乎（图文B）
✅ 什么值得买（图文B）
✅ B站（视频）
✅ 抖音视频（视频）
✅ 淘宝/京东（商品页）

每个平台取前10-20条结果
记录到表3
```

### **步骤3：提取链接内容（关键）**
```
对表3中的每个链接：

1. 访问链接
2. 根据内容格式，提取结构化内容：
   - 图文A → 提取标题、正文、图片、评论
   - 图文B → 提取标题、内容流（文本+图片有序）、评论
   - 视频-有字幕 → 提取标题、简介、字幕、评论
   - 视频-无字幕 → 提取标题、简介、语音转文本、评论
   - 商品页 → 提取标题、卖点、参数、详情、评价
   
3. 图片处理：
   - 记录图片URL
   - 生成图片描述（AI描述：如"耳机外观图"）
   - 识别图片主题（如"产品展示"/"使用场景"）
   - 暂不提取图片文字
   
4. 组织成JSON格式
5. 写入表4
```
