## 二、滑块验证码

### 2.1 识别逻辑（Rust + OpenCV）

**输入**: 背景图 base64、滑块拼图 base64
**输出**: `{ x, y }` 缺口左上角坐标（像素）

**核心算法**:

1. **加载图像**
   - 背景图：灰度模式（用于边缘检测）
   - 滑块拼图：原图（用于获取参考尺寸 `tpl_w`, `tpl_h`）

2. **预处理**
   - 高斯模糊（3×3 核）降噪
   - Canny 边缘检测（阈值 100, 200）

3. **轮廓查找**
   - `find_contours` 仅查找最外层轮廓（`RETR_EXTERNAL`）

4. **过滤与匹配**
   - 排除宽度/高度小于滑块 80% 的轮廓
   - 排除宽高比异常的细长轮廓
   - 排除背景图左侧 1/5 区域（滑块起始区）
   - 选择与滑块尺寸差异最小的轮廓作为缺口

5. **输出**
   - 返回缺口矩形左上角 `rect.x`, `rect.y`

**完整代码**:

```rust
pub fn solve_captcha_opencv(bg_b64: &str, target_b64: &str) -> Result<OutputData> {
    // 1. Load images
    let bg_mat = decode_base64_to_mat(bg_b64, IMREAD_GRAYSCALE)?;
    let target_mat = decode_base64_to_mat(target_b64, IMREAD_UNCHANGED)?;
    let tpl_w = target_mat.cols();
    let tpl_h = target_mat.rows();

    // 2. Preprocess: Gaussian blur -> Canny edge detection
    let mut blurred = Mat::default();
    gaussian_blur(&bg_mat, &mut blurred, Size::new(3, 3), 0.0, 0.0, BORDER_DEFAULT, AlgorithmHint::ALGO_HINT_DEFAULT)?;
    let mut edges = Mat::default();
    canny(&blurred, &mut edges, 100.0, 200.0, 3, false)?;

    // 3. Find contours
    let mut contours = VectorOfVectorOfPoint::new();
    find_contours(&edges, &mut contours, RETR_EXTERNAL, CHAIN_APPROX_SIMPLE, Point::new(0, 0))?;

    // 4. Filter contours to find the real gap
    let mut best_rect: Option<Rect> = None;
    let mut min_size_diff = i32::MAX;
    let size_tolerance_ratio = 0.20;
    let min_w = (tpl_w as f32 * (1.0 - size_tolerance_ratio)) as i32;
    let min_h = (tpl_h as f32 * (1.0 - size_tolerance_ratio)) as i32;

    for i in 0..contours.len() {
        let contour = contours.get(i)?;
        let rect = bounding_rect(&contour)?;
        if rect.width < min_w || rect.height < min_h { continue; }
        if rect.x < (bg_mat.cols() / 5) { continue; }  // Exclude left start area
        let total_diff = (rect.width - tpl_w).abs() + (rect.height - tpl_h).abs();
        if total_diff < min_size_diff {
            min_size_diff = total_diff;
            best_rect = Some(rect);
        }
    }

    if let Some(rect) = best_rect {
        Ok(OutputData { x: rect.x, y: rect.y, score: 1.0 - (min_size_diff as f32 / 100.0) })
    } else {
        Ok(OutputData { x: bg_mat.cols() / 2, y: bg_mat.rows() / 2, score: 0.0 })
    }
}
```

**Node.js 调用封装**: `src/utils/slide-solver.js`

```javascript
const { spawn } = require('child_process');
const path = require('path');
const fs = require('fs');

function getSolverBinaryPath() {
  const binaryName = process.platform === 'win32' ? 'slide-solver.exe' : 'slide-solver';
  return path.join(__dirname, '..', 'lib', binaryName);
}

function solveCaptcha(bgBuffer, targetBuffer) {
  return new Promise((resolve, reject) => {
    const binPath = getSolverBinaryPath();
    if (!fs.existsSync(binPath)) {
      return reject(new Error(`Slide solver binary not found at: ${binPath}`));
    }
    const inputData = JSON.stringify({
      bg_base64: bgBuffer.toString('base64'),
      target_base64: targetBuffer.toString('base64'),
    });
    const rustProcess = spawn(binPath, [], { cwd: path.dirname(binPath) });
    let resultString = '';
    let errorString = '';
    rustProcess.stdout.on('data', (data) => { resultString += data.toString(); });
    rustProcess.stderr.on('data', (data) => { errorString += data.toString(); });
    rustProcess.on('close', (code) => {
      if (code !== 0) return reject(new Error(`Slide solver failed (exit ${code}): ${errorString}`));
      try {
        resolve(JSON.parse(resultString));
      } catch (e) {
        reject(new Error(`Invalid JSON from slide solver: ${resultString}`));
      }
    });
    rustProcess.on('error', (err) => reject(new Error(`Failed to start: ${err.message}`)));
    rustProcess.stdin.write(inputData);
    rustProcess.stdin.end();
  });
}
```

### 2.2 拖拽逻辑（人类化滑动）

**流程**（`solve()` 方法串联）:

1. 元素选择器：`img.captcha-verify-image`（背景）、`img.captcha-verify-image-slide`（滑块）
2. `bgHandle.screenshot({ type: 'png' })`、`targetHandle.screenshot({ type: 'png' })` 得到 Buffer
3. `solveCaptcha(bgBuffer, targetBuffer)` 调用 Rust，得到 `rustResult.x`
4. 拖拽距离：`dragDistance = rustResult.x - (sliderPieceBox.x - bgBox.x)`
5. 起点：`.captcha-slider-btn` 中心；若无则用滑块图中心
6. `humanLikeDrag(actualPage, startX, startY, dragDistance)`

**humanLikeDrag 实现**（完整可运行）:

```javascript
function easeOutCubic(t) {
  return 1 - Math.pow(1 - t, 3);
}

async humanLikeDrag(page, startX, startY, distance) {
  const dir = distance >= 0 ? 1 : -1;
  const absDist = Math.max(Math.abs(distance), 10);
  const totalTime = 900 + Math.random() * 400;   // 900-1300ms
  const steps = 30 + Math.floor(Math.random() * 20);  // 30-50 步

  let prevX = startX;
  let prevY = startY;

  await page.mouse.move(startX, startY);
  await page.mouse.down();

  const startTime = Date.now();
  for (let i = 1; i <= steps; i++) {
    const t = i / steps;
    const eased = easeOutCubic(t);
    const targetX = startX + dir * absDist * eased;
    const jitterY = (Math.random() - 0.5) * 4;
    const nextY = startY + jitterY;
    const dx = targetX - prevX;
    const dy = nextY - prevY;

    await page.mouse.move(prevX + dx, prevY + dy, { steps: 2 });
    prevX = prevX + dx;
    prevY = prevY + dy;

    const elapsed = Date.now() - startTime;
    const remaining = totalTime - elapsed;
    const remainingSteps = steps - i;
    const sleep = remainingSteps > 0 ? Math.max(5, remaining / remainingSteps) : 0;
    if (sleep > 0) await page.waitForTimeout(sleep);
  }

  const adjust = 3 + Math.random() * 3;
  await page.mouse.move(prevX - dir * adjust, prevY, { steps: 3 });
  await page.waitForTimeout(80 + Math.random() * 120);
  await page.mouse.move(prevX, prevY, { steps: 2 });
  await page.waitForTimeout(120 + Math.random() * 180);
  await page.mouse.up();
}
```

**关键点**:
- 使用 `easeOutCubic` 实现末尾减速，模拟真人滑动
- Y 轴加入 ±2px 随机抖动
- 末尾过冲 3–6px 再回正，增加真实感

## 三、图片选择验证码

### 3.1 识别逻辑（火山引擎 Vision LLM）

**输入**: 问题文本、图片 base64、图片索引
**输出**: `{ imageIndex, isMatch, rawAnswer }`

**模型**: `doubao-vision-pro-32k`（火山引擎）

**Prompt 设计**:

```
System: You are an expert at analyzing images and answering visual reasoning questions.
        Provide clear, concise yes/no answers about whether an image matches the given criteria.

User:   Question: "{question}"
        Does this image (Image #N) match the question? Answer with:
        1. YES or NO
        2. Brief explanation (one sentence)
        Format: YES/NO | explanation

        [image_url: data:image/jpeg;base64,...]
```

**解析逻辑**: 响应内容包含 `YES` 即判定为匹配

**完整代码**（含 https 请求）:

```javascript
const https = require('https');

async function analyzeImageWithLLM(question, imageBase64, imageIndex) {
  const endpoint = process.env.VOLCENGINE_ENDPOINT || 'https://ark.cn-beijing.volces.com/api/v3';
  const apiKey = process.env.VOLCENGINE_API_KEY || '';
  const model = process.env.VOLCENGINE_MODEL || 'doubao-vision-pro-32k';

  const requestBody = JSON.stringify({
    model,
    messages: [
      { role: 'system', content: 'You are an expert at analyzing images and answering visual reasoning questions. Provide clear, concise yes/no answers about whether an image matches the given criteria.' },
      {
        role: 'user',
        content: [
          { type: 'text', text: `Question: "${question}"\n\nDoes this image (Image #${imageIndex + 1}) match? Answer with:\n1. YES or NO\n2. Brief explanation\n\nFormat: YES/NO | explanation` },
          { type: 'image_url', image_url: { url: `data:image/jpeg;base64,${imageBase64}` } }
        ]
      }
    ],
    temperature: 0.3,
    max_tokens: 200
  });

  const url = new URL(endpoint + '/chat/completions');
  const options = {
    hostname: url.hostname,
    port: 443,
    path: url.pathname,
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${apiKey}`,
      'Content-Length': Buffer.byteLength(requestBody)
    }
  };

  return new Promise((resolve, reject) => {
    const req = https.request(options, (res) => {
      let data = '';
      res.on('data', (chunk) => { data += chunk; });
      res.on('end', () => {
        try {
          const response = JSON.parse(data);
          const answer = response.choices?.[0]?.message?.content || '';
          const isMatch = answer.toUpperCase().includes('YES');
          resolve({ imageIndex, isMatch, rawAnswer: answer, confidence: isMatch ? 'high' : 'low' });
        } catch (e) {
          reject(new Error(`Parse error: ${e.message}`));
        }
      });
    });
    req.on('error', (e) => reject(new Error(`API failed: ${e.message}`)));
    req.setTimeout(30000, () => { req.destroy(); reject(new Error('Timeout')); });
    req.write(requestBody);
    req.end();
  });
}
```

### 3.2 拖拽逻辑（贝塞尔曲线 + 鼠标拖拽）

**贝塞尔路径生成**:

```javascript
function generateBezierPath(x0, y0, x3, y3, steps = 50) {
  const points = [];
  const dx = x3 - x0;
  const dy = y3 - y0;
  const x1 = x0 + dx * (0.2 + Math.random() * 0.3) + (Math.random() - 0.5) * 100;
  const y1 = y0 + dy * (0.2 + Math.random() * 0.3) + (Math.random() - 0.5) * 100;
  const x2 = x0 + dx * (0.5 + Math.random() * 0.3) + (Math.random() - 0.5) * 100;
  const y2 = y0 + dy * (0.5 + Math.random() * 0.3) + (Math.random() - 0.5) * 100;

  for (let i = 0; i <= steps; i++) {
    const t = i / steps;
    const t1 = 1 - t;
    const x = t1 * t1 * t1 * x0 + 3 * t1 * t1 * t * x1 + 3 * t1 * t * t * x2 + t * t * t * x3;
    const y = t1 * t1 * t1 * y0 + 3 * t1 * t1 * t * y1 + 3 * t1 * t * t * y2 + t * t * t * y3;
    points.push({ x, y });
  }
  return points;
}
```

**拖拽到目标区**:

```javascript
async dragImagesToTarget(imageIndices) {
  const canvasElements = await this.pageOrFrame.$$('canvas');
  const dragArea = await this.pageOrFrame.$('.drag-area');
  const dragAreaBox = await dragArea.boundingBox();
  const actualPage = this.getActualPage();

  for (const index of imageIndices) {
    const canvas = canvasElements[index];
    const canvasBox = await canvas.boundingBox();
    const sourceX = canvasBox.x + canvasBox.width / 2;
    const sourceY = canvasBox.y + canvasBox.height / 2;
    const targetX = dragAreaBox.x + dragAreaBox.width / 2 + (Math.random() - 0.5) * 20;
    const targetY = dragAreaBox.y + dragAreaBox.height / 2 + (Math.random() - 0.5) * 20;

    const pathPoints = this.generateBezierPath(sourceX, sourceY, targetX, targetY, 50);

    await actualPage.mouse.move(sourceX, sourceY);
    await actualPage.waitForTimeout(100 + Math.random() * 100);
    await actualPage.mouse.down();
    for (let i = 0; i < pathPoints.length; i++) {
      await actualPage.mouse.move(pathPoints[i].x, pathPoints[i].y);
      await actualPage.waitForTimeout(2 + Math.random() * 3);
    }
    await actualPage.mouse.up();
    await actualPage.waitForTimeout(300 + Math.random() * 400);
  }
}
```

**关键点**:
- 起点：canvas 中心
- 终点：`.drag-area` 中心 ±10px 随机偏移
- 使用三次贝塞尔曲线生成路径，控制点带随机扰动
- 每步 2–5ms 间隔，模拟人类拖拽速度

```javascript
async function solveCaptchaAuto(pageOrFrame) {
```

2. 若存在，遍历 `page.frames()` 找 `frame.url().startsWith('https://rmc.bytedance.com/verifycenter/captcha/v2')`

```rust
use base64::prelude::*;
use opencv::{
    core::{Mat, Point, Rect, Size, Vector, BORDER_DEFAULT, AlgorithmHint},

    prelude::*,

use serde::{Deserialize, Serialize};
use std::io::{self, BufRead, Write};

type VectorOfVectorOfPoint = Vector<Vector<Point>>;

#[derive(Deserialize)]
struct InputData { bg_base64: String, target_base64: String }

#[derive(Serialize)]
struct OutputData { x: i32, y: i32, score: f32 }

fn decode_base64_to_mat(b64: &str, flags: i32) -> Result<Mat> {
    let bytes = BASE64_STANDARD.decode(b64).context("decode base64")?;
    Ok(imdecode(&Vector::from_slice(&bytes), flags)?)
}

fn main() -> Result<()> {
    let mut json_str = String::new();
    io::stdin().lock().read_line(&mut json_str)?;
    let input: InputData = serde_json::from_str(&json_str)?;
    let output = solve_captcha_opencv(&input.bg_base64, &input.target_base64)?;
    print!("{}", serde_json::to_string(&output)?);
    io::stdout().flush()?;
    Ok(())
}
// solve_captcha_opencv 见 2.1 节
```

## 八、Web采集异常处理顺序（运行策略）

Web-only 采集时建议严格按以下顺序处理，避免静默失败：

1. **先检查默认对话框可输入**
   - 输入框必须可见、可编辑、未被弹窗遮挡；
   - 若未就绪，先尝试点击“新对话/新建对话”再重试。
2. **若命中验证码**
   - 优先调用本文档的自动求解流程（滑块 / 图片选择）。
3. **自动求解失败后切换账号**
   - 标记当前手机号 busy，重新走登录流程换号。
4. **仍失败时弹窗人工处理**
   - 必须切到 headed 模式，等待人工完成验证/登录。
5. **超时后结束进程**
   - 记录终态日志（阶段、是否验证码、是否登录、输入框状态），避免无限挂起。
